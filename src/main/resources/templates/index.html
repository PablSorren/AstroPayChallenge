Bienvenidos!

  1. '/posts?pageNumber=X&pageSize=Z' para ver los posts de forma paginada (el pageNumber inicial = 0)

    2. ej.: http://localhost:8080/posts?pageNumber=0&pageSize=3

  2. '/posts/id_Post' para ver un post con el ID especificado.

    3. ej.: http://localhost:8080/posts/100

  3. '/comments/post id' para ver todos los comentarios asociados al Post con el ID especificado.

    4. ej.: http://localhost:8080/comments/9

  4. '/posts/title?word=palabra' para filtrar los posts cuyo titulo contengan la 'palabra' buscada

    5. ej.: http://localhost:8080/posts/title?word=iusto

  5. '/posts' para ver todos los posts

  6. '/comments' para ver todos los comentarios.
